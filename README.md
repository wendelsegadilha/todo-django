# Lista de Tarefas com Python e Django

### Descrição
Projeto simples de uma lista de tarefas com Python e Django que explora as funcionalidades básicas do framework e da linguagem de programação. 


### Funcionalidades
- Cadastro
- Consulta
- Atualização
- Exclusão

### Tecnologias utilizadas: 
- HTML
- CSS
- JavaScript
- Pytho
- Django
- SQLite
